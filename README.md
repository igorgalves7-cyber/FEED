# Projeto Social Feed - Expo

Este projeto é um simulador de rede social desenvolvido em React Native com Expo.

## Funcionalidades
- **Feed:** Visualização de postagens e criação de novos posts usando links de imagem da web.
- **Perfil:** Página de usuário com banner, avatar e grid de fotos das postagens realizadas.

## Como rodar
1. Instale as dependências: `npm install`
2. Inicie o projeto: `npx expo start`
